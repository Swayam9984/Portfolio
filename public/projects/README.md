# Project Images

Add your project screenshots here:

- `social-media.jpg` - Screenshot of Social Media Website
- `amazon-clone.jpg` - Screenshot of Amazon Clone
- `dsa-instructor.jpg` - Screenshot of DSA Instructor
- `religious-app.jpg` - Screenshot of Religious App

## How to Add Your Own Images

1. Take screenshots of your projects
2. Save them in this `public/projects/` folder
3. Update the `image` field in `components/Projects.tsx`:
   - Use `/projects/your-image.jpg` for local images
   - Or use full URL for external images

## Image Recommendations

- Size: 800x400px or 16:9 aspect ratio
- Format: JPG or PNG
- File size: Under 500KB for better performance
- Quality: High resolution for crisp display
